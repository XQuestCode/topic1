IoT applications work with IoT devices to perform variety of tasks like edge computing, surveillance, generate real time insights from data collected.
Some of the features needed to be supported are: Secured Communication, Cross platform, Interactive and user friendly UI, Seamless computing, and Data Security.
Some examples are: Smart home or Health monitoring wearables

A smart building involves the installation and use of advanced and integrated building technology systems.
These systems include building automation, life safety, telecommunications, user systems, and facility management systems.

In this context, the Control Interface might be an G.U.I (could be an webpage, application) which allows users to access or make changes, customize the features available through the technology.
Some of them could be Doors security on basis of credentials, Automated or Manual Curtains control depending on light intensity, or Automated home cleaning.
